Stickynotes Application

Author: Mpho Kgooa
Framework: Django
Task: CRUD Web Application – Sticky Notes

📌 Project Overview

The Stickynotes Application is a Django-based web application that allows users to create, read, update, and delete notes.
It was built to demonstrate understanding of:

Django project and app structure

URL routing and naming conventions

Views and CRUD functionality

Form handling

Templates and template inheritance

Testing with Django’s built-in test framework

This project is lightweight, fully functional, and follows clean Django development principles.

🎯 Features
✔ Create Notes

Users can add new notes using a simple and user-friendly form.

✔ View Notes

All notes are displayed on a clean dashboard with titles, previews, and timestamps.

✔ Update Notes

Notes can be edited at any time using the update form.

✔ Delete Notes

Notes can be permanently removed through a confirmation page.

✔ Responsive UI

The application uses Bootstrap styling for an improved user experience.

✔ Full Test Coverage

The included test suite covers:

creating a note

reading notes

updating a note

deleting a note

All tests pass successfully.

🛠️ Tech Stack

Python 3.x

Django 5.x

Bootstrap 5

SQLite (default development database)
